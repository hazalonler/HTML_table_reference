HTML TABLE REFERENCE

Tag Table

Tag Name Tag Format Description of Tag
Table <table> The <ul>element create a table that contains all of the tabular data we plan on displaying.
Table Head <thead> The <thead>element includes the table headings. Note that the table's head still requires a row in order to contain the table headings.
Table Rows <tr> The <tr> allows to add rows to enter table data.
Table Data <td> The <td> provides to enter data into rows
Table Headings <th> The <th> element is used to add titles to rows and columns. And also, the scope attribute which can take row or col values allows to make it clear that which heading is assigned to which row or column.

Attribute Table

Attribute Name Attribute Format Description of Attribute
Column Span colspan colspan attribute describes how many columns a table data can span
Row Span rowspan rowspan attribute describes how many rows a table data can span.
